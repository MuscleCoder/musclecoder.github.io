mybatis-collection-with-only-size-one

Table A
| id | a_1 | a_2|
|—|—|—|
|1 | aaa | bbb|

Table B
| id | a_id | b_1 | b_2|
|—|—|—|—|
| 1 | 1 | ccc| ddd|
| 2 | 1 | eee| fff|

The result only contains a collection with size 1.

After check this is because I miss to config b.id as bid, and then the a.id is regard as the id of table B.
